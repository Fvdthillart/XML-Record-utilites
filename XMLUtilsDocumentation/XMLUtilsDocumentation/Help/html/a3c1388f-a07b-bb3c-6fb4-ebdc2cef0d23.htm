Remarks

This class processes a csv, anonymizes according to the rules in anonymizeCSV.txt and writes the result to a file with the orginal filename and "_Anonymous" appended to it This is static class with a static main method to provide a commandline interface for the AnonymizerCSV class.

It evaluates the commandline and translates the parameters to arguments voor the constructor of AnonymizerCSV

Parameters are:

  1. form:<any name>.csv - Name of the CSV file
  2. form:seperator="<any single character>" - Indicates the character that separates the values
  3. form:textindicator=<any single character> - Indicates the start and end of a text string
  4. form:headerrow=Y or N - Indicates if there's a headerrow
Examples

Example for a call to the executable:

Copy
AnonymizeCSV.exe test.csv seperator=";" textIndicator=" headerrow=N
