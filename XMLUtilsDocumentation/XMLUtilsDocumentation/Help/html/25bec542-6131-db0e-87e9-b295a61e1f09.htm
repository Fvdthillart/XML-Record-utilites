Core.XMLRecords Namespace
The XMLRecords namespace contains all the classes that work with the concept of XML Records. An XMLRecord is an XML subtree which has a containertag as root element and is identified by 1 or more identifying elements. The concept was conceived of when a large XML file couldn't be compared and a way was needed to make this possible.
