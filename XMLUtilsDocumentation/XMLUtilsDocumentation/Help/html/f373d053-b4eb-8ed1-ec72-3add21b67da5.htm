Remarks

This class anonymizes an XML file according to the rules in Anonymize.txt and writes the result to a file with the orginal filename and "_Anonymous" appended to it This is a static class with a static main method to provide a commandline interface for the AnonymizerXML class.

It evaluates the commandline and translates the parameters to arguments for the constructor of the AnonymizerXML class

Parameters are:

  1. <any name>.xml - Name of the file to be anonymized
  2. any XML element(s) without < and > - The element(s) that define(s) the subtree(s) to be anonymized
