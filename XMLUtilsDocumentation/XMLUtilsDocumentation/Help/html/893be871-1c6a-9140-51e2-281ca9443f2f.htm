Remarks

This class splits an XML file into the subtrees of an element called the containertag and writes each XML subtree to a directory with the same name as the XML file to be split.
The filenames in the directory are a concatenation of the names and values of elements, determined by socalled identifying tags which together have to uniquely identify the XMLTree otherwise files are overwritten. This is a static class with a static main method to provide a commandline interface for the XMLRecordFile class.

It evaluates the commandline and translates the parameters to arguments for the constructor of the XMLRecordFile class

Parameters are:

  1. <any name>.xml - Name of the XML file to be split
  2. any XML element without < and > - The element that defines the subtree to be anonymized
  3. any XML element without < and > in the subtree identified by the second parameter - The element(s) that define(s) a unique ID for the subtree
