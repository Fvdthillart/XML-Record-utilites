The AnonymizeRule type exposes the following members.

Properties
  Name Description
Allowed
Indicates whether a rule is Allowed to be used. This attribute allows for disabling a rule during processing if deemed necessary
Filter
Read only property that returns the filter of the rule
FilterLocation
First part of the filter. Contains the data element that's being filtered
FilterOperator
Second part of a filter. Contains the operator of the filter like <
FilterValue
Third part of a filter. Contains the value to which the filter compares the data element, specified in FilterLocation
IsFiltered
Indicates that the rule has a filter
Location

Location of the value to be anonymized.

This can be an xpath expression for XML or the index number of a field in a CSV file
Method
Contains the method. See AnonymizeRuleMethodType for more information
