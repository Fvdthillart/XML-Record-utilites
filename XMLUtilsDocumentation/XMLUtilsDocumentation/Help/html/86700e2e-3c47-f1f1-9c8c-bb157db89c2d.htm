AnonymizeCSV
This class anonymizes a csv according to the rules in AnonymizeCSV.txt and writes the result to a file with the orginal filename and "_Anonymous" appended to it This is static class with a static main method to provide a commandline interface for the AnonymizerCSV class.

Executable: AnonymizerCSV.exe
