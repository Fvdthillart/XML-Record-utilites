Core.Anonymization Namespace
This namespace contains all the anonymization classes. The root class contains all the anonymization functions without file handling. The descendants add functionality to anonymize a specific type of file according to a set of rules
Classes
  Class Description
AnonymizeCache
This class contains an original value and its corresponding anonymized value so that one way anonymizations can still have the same result in the same file. It's compatible with the List<T> datatype
Anonymizer
The anonymizer class is used to replace meaningful values with meaningless values. Anonymizing is done to cloak personal information like names, account number and anything that can be used to identify a person and use that information in a way that was not intended by the person, identified by the information
AnonymizerCSV
This class anonymizes a CSV file, based on a file with Anonymization rules
AnonymizeRule

This class is a value class for storing anonymization rules

An anonymization rule consists of a location and a method of anonymization as specified in AnonymizeRuleMethodType.

AnonymizerWithRules
Extension of the Anonymizer class to handle files that contain Anonymizer rules
